# Reproductor de Música

Pequeña aplicación de reproducción de audio en Flutter creada como proyecto personal.

## Características
- Reproducción de archivos de audio locales
- Manejo de sesiones de audio y notificaciones
- Gestión de listas de reproducción
- Interfaz oscura (tema por defecto)

## Requisitos
- Flutter
- Android SDK / Xcode
- Java y herramientas de compilación para Android

## Construir APK (Android)
```bash
flutter build apk --release
```

## Permisos importantes
Esta app requiere permisos de almacenamiento y audio en Android. Asegúrate de aceptar los permisos cuando se soliciten o añadirlos en `AndroidManifest.xml` si personalizas el proyecto.
